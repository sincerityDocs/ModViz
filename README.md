# ModViz - Virtual Car Mod Visualizer & Marketplace

## Business Overview
Premium e-commerce platform for car modifications (vinyl wraps, splitters, spoilers, carbon fiber aero, etc.) focused on BMW, Audi, Mercedes-Benz, Lamborghini, Nissan Z/GTR, Toyota Supra/Camry/86, Subaru BRZ, Honda Accord, and select sports models (2018+).

**Core Differentiator**: Ultra-fast AI image generation that visualizes mods on customer’s exact car from multiple angles.

## Supported Vehicles
- BMW (all)
- Audi (all)
- Mercedes-Benz (all)
- Lamborghini (all)
- Nissan: New Z, 350Z, GT-R
- Toyota: Camry (all trims), Supra, 86
- Subaru BRZ (2016+)
- Honda Accord (all)
- Sports-oriented SUVs from above brands only
- Select other popular sports cars

## Project Structure
- `/agents/` - Agent workflows and prompts
- `/website/` - Frontend/backend code (Next.js + Shopify)
- `/sourcing/` - Supplier data and scripts
- `/marketing/` - Content and campaign assets
- `CTO_PROMPT.md` - Master prompt for cto.new

## Quick Start
1. Use the CTO_PROMPT.md on cto.new to bootstrap agents.
2. Focus on premium/name-brand sourcing (Seibon, Vorsteiner, APR, Vivid Racing, etc.) with ~25% OTD margins.
3. Prioritize visualization engine.

## Goals
- Fast MVP launch
- High-converting visualization experience
- Balanced sourcing: name-brand quality + realistic margins
