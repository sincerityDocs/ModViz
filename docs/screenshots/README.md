# ModViz MVP Preview Screenshots

This directory contains visual previews of the ModViz car modification visualizer and e-commerce web platform in action:

1. **`modviz-home.png`**: The interactive visualizer landing stage, showing the car model, modification categories, and stock/custom asset selectors.
2. **`modviz-wrap-selected.png`**: A preview of the interactive car vinyl wrap customizer.
3. **`modviz-compare.png`**: The interactive sliding split-screen showing before (OEM paint/spec) and after (modded/wrapped spec) custom composite renders.
4. **`modviz-checkout.png`**: The pre-launch hardened simulated mock checkout and authorized order confirmation flow.

---

### Important Notice
These screenshots depict a **work-in-progress pre-launch MVP demo**. They show simulated configurations, mock order checkouts, and pre-launch supplier pricing structures for the purpose of visual and technical evaluation. They do **not** represent approved production or live-commerce catalog details, and all fulfillment, drop-shipping, and supplier connections are strictly simulated.
